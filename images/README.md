# Profile and Brand Images

## Favicon and App Icons
- Create a personalized favicon using [Favicon.io](https://favicon.io/favicon-generator/)
- Generate a full set of icons including `favicon.ico` and `apple-touch-icon.png`
- Use your initials "AM" on a gradient background matching your site colors

## Profile Photo
- For `profile.jpg`, use a professional headshot
- Dimensions: 1200x1200px (square)
- Keep the file size under 200KB for fast loading
- Use good lighting and a neutral background
